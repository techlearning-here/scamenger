Hero section redesign — The stock photo is generic. Replace with a bold illustration or icon-driven hero that immediately communicates "report → get help → stay safe."
Card-based scam categories — Replace the flat link list with visual cards featuring icons (🎣 phishing, 💳 bank fraud, 💘 romance). Cards are more scannable and tappable on mobile.
📱 Mobile & Accessibility

Country auto-detection — Use navigator.language or IP geolocation to pre-select the country, reducing friction.
🧭 Navigation & Flow

Breadcrumb trail — On inner pages, show the path (Home → US → Phishing) so users can backtrack easily.
Progress indication — After selecting a scam type, show a clear "what happens next" flow: Report → Track → Prevent.
